# Michi v7 — scope and unresolved gaps

This file turns the project source-of-truth into a locked build target for the v7 branch.

## What v7 must materially improve

### 1. Content engine
- move beyond demo-scale content
- support a real path from N5/N4 toward N2 readiness
- include core scenes, review, transfer, academic, design, reality, and path content families
- reduce dependence on live AI

### 2. Design / Study in Japan axis
- design vocabulary
- critique language
- presentation language
- academic comprehension patterns
- university-facing preparation logic

### 3. Path to Japan
- more practical and less decorative
- stronger reality cards
- clearer now / next / later structure
- better factual trust layer

### 4. Visual system
- Real Japan / Dream Japan / Process Japan roles
- stronger tagging and usage rules
- more explicit scene/path/design mappings

### 5. Chat layer
- factual mode
- guide mode
- sources block
- follow-up prompts
- clearer verification states

### 6. Learning mechanics
- softer re-entry after pauses
- review and reinforcement
- progression support
- ADHD-friendly task framing

## Gaps still to close on this branch
- large-scale static content expansion is not yet fully committed
- stronger academic/design content families still need to be written into repo
- some UI screens still need to be updated to surface the larger content system
- final release audit against project notes still needs to be done before PR

## Rule for this branch
Do not call this finished until the branch clearly moves beyond patch-level recovery.
