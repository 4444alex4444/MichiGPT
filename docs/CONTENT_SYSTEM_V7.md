# Michi Content System v7

This document locks the intended content architecture for the v7 branch.

## 1. Product-level content promise
Michi should no longer feel like a beautiful prototype with a chat patch. The static content layer must be large enough to support a real path from N5/N4 toward N2 readiness with strong study-in-Japan and design relevance.

## 2. Major content families

### Core scenes
Atmospheric scenes remain the emotional doorway into the product, but they are no longer the whole product.

### Review cards
Short low-pressure cards for recall, re-entry, and reinforcement.

### Transfer cards
Cards that move language from scene recognition into practical expression, action, comparison, planning, and study use.

### Academic Japanese
Cards for instructions, note-taking, lecture flow, assignments, discussions, and presentations.

### Design Japanese
Cards for visual description, critique, rationale, composition, hierarchy, typography, materials, and portfolio explanation.

### Reality cards
Cards that make Path to Japan practical and less bureaucratic: next-step framing, financial realism, support systems, documents, housing, and study adaptation.

### Path cards
Longer practical orientation blocks for future-in-Japan direction.

## 3. Curriculum direction
The v7 branch should follow a two-year logic:
- foundation_n5_n4
- bridge_n3
- academic_readiness_n2

The branch already includes a monthly scaffold to structure this path.

## 4. ADHD-friendly implementation rules
- low-friction entry
- visible finish lines
- bounded tasks
- clear prompts
- no shame after pauses
- strong re-entry support
- multiple useful response modes

## 5. Design / study-in-Japan requirement
Design Japanese must not remain decorative. It should include:
- critique language
- presentation language
- process explanation
- portfolio and studio talk
- design-specific academic survival language

## 6. Final branch rule
The v7 branch is not finished until it clearly moves beyond patch-level recovery and becomes a materially stronger static foundation.
